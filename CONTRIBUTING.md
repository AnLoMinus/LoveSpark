# CONTRIBUTING — תרומה ל-LoveSpark

תודה שבחרתם לתרום! אנחנו אוהבים תוכן עשיר, רגיש ומכבד.

## עקרונות
- כתיבה בעברית ברורה, נגישה ומכבדת.
- כל קלף חייב להיות ייחודי בנימה ובכוונה.
- אין להוסיף תוכן פוגעני, מפלה או פולשני.

## זרימת עבודה
1. צרו Fork והקימו סביבה מקומית.
2. פתחו ענף חדש בשם ברור (`feature/cards-emotion`).
3. ערכו או הוסיפו קלפים ב-`index.html` ובמידת הצורך ב-`assets`.
4. ודאו שהעיצוב נשאר רספונסיבי.
5. פתחו Pull Request עם הסבר קצר.

## קומיטים
אנו משתמשים ב-Conventional Commits:
- `feat:` תוספת פיצ'ר או קלף חדש.
- `fix:` תיקון ניסוח או באג.
- `docs:` עדכוני תיעוד.

דוגמה:
```
feat: הוספת קלף "מפה סודית"
```

## בדיקות מקומיות
פתיחת `index.html` בדפדפן מספיקה לאימות חזותי בסיסי.
