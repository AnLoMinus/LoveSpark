# CODE OF CONDUCT — קוד התנהגות

אנחנו מחויבים לסביבה בטוחה, מכבדת ומכילה.

## הציפיות שלנו
- לדבר בכבוד וללא שיפוטיות.
- להקשיב ולשאול שאלות מתוך סקרנות.
- לכבד גבולות אישיים ותרבותיים.

## התנהגות לא מקובלת
- הטרדה, השפלה או איום מכל סוג.
- שפה פוגענית או מינית בוטה.
- שיתוף מידע אישי ללא הסכמה.

## אכיפה
במקרה של פגיעה, ניצור קשר, נבהיר את הבעיה ונפעל בהתאם. במידת הצורך נחסום גישה לפרויקט.
