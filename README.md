# LoveSpark — בית חם למשחקים זוגיים בעברית

LoveSpark הוא מארז קלפים ומשימות לשיחה זוגית שמחזיר את הניצוץ בעדינות, בהומור ובקרבה.
הפרויקט נבנה כדי לספק חוויה קצרה, נגישה ואותנטית לזוגות בכל שלב.

## חזון
- ליצור רגע יומיומי של חיבור אמיתי.
- לשלב משחקיות, עומק והקשבה בלי לחץ.
- להציע תוכן עברי עשיר ואסתטי.

## שימוש מהיר
1. בחרו קלף אחד מתוך החבילה.
2. קראו בקול וענו בתורות.
3. סיימו במחווה קטנה: חיבוק, הודעה או רגע שקט.

## תיעוד מקוצר
- `index.html` — האתר הראשי עם כל הקלפים וההכוונה.
- `ROADMAP.md` — לאן אנחנו מכוונים בהמשך.
- `CONTRIBUTING.md` — איך תורמים קלפים ותוכן.
- `CHANGELOG.md` — יומן גרסאות לפי SemVer.
- `SECURITY.md` — דיווח על חולשות והתחייבויות.

## קישורים מהירים
- מדיניות קהילה: `CODE_OF_CONDUCT.md`
- תמיכה ושאלות: `SUPPORT.md`
- מודל ניהול: `GOVERNANCE.md`
- רישוי: `LICENSE`

## קרדיטים
תודה לכל מי שתרם רעיונות, ניסוחים, השראה ועיצוב — במיוחד לשיתופי הקהילה שהובילו לבניית קלפים איכותיים בעברית.
הקרדיטים המלאים נמצאים ב-`ACKNOWLEDGEMENTS.md`.
